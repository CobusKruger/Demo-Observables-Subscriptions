Subscribe/Unsubscribe

The observable will emit a prime number every second and this page will show all subscribers that it is aware of.

  • Each subscriber will list its prime numbers on the page.
  • Each subscriber will write to the console each time it receives a value.
  • Add or remove subscribers by clicking the corresponding buttons.
  • Complete the observable to stop all subscriptions from receiving any more values.
  • Navigate back to home to destroy this component. Come back here and check Chrome Developer Tools to verify if the subscriptions are still active.
